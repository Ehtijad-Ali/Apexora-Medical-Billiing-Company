# 🏥 Apexora Medical Billing Company

![Apexora Banner](https://images.unsplash.com/photo-1576091160550-2173dba999ef?auto=format&fit=crop&q=80&w=1000)

Apexora is a premium, high-performance web application designed for a modern medical billing company. Built with a focus on **precision, transparency, and revenue optimization**, this platform serves as a comprehensive portal for healthcare providers to explore full-service Revenue Cycle Management (RCM) solutions.

---

## 🚀 Key Features

### 💎 Premium User Experience
- **Dynamic Navigation:** Smooth scrolling and active state tracking with a glassmorphism Navbar.
- **Interactive Savings Calculator:** A functional tool for practices to estimate ROI, featuring a unique "Practice Health Score" visualization.
- **Executive Team Bios:** A sophisticated, modal-based team section showcasing leadership expertise with LinkedIn and Email integration.
- **Advanced Animations:** Powered by `framer-motion` for smooth section transitions, hover effects, and parallax backgrounds.

### 🛠️ Technical Excellence
- **Responsive Design:** Mobile-first architecture ensuring a flawless experience from smartphones to ultra-wide monitors.
- **Specialty-Specific Solutions:** Detailed breakdown of 75+ medical specialties supported.
- **HIPAA Focused:** Design and content tailored to emphasize security and federal compliance.
- **Real-time Interaction:** Integrated WhatsApp floating button for instant client communication.

## 🚦 Getting Started

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
   ```bash
   npm run dev
   ```
The app will be available at `http://localhost:3000`.

### Build
Generate a production-ready build:
   ```bash
   npm run build
   ```
